Installation
```
chmod -R 755 ./main.sh
```

USAGE
```
./main.sh {{title}} {{template name}} {{url}} {{country code}} {{language}}
```

Ex.
```
./main.sh NewFile new-file-template new-file-url th en
```

and then copy result.json to mock.model.json in level :children

$1 = title 

$2 = template name

$3 = url

$4 = country code

$5 = language
